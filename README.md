# Phrase

##About Phrase

Phrase is an annotation product for teachers, classrooms, and students.


## Usage

###For Development
```
cd tnw
npm install
npm run dev
```

Runs on http://localhost:9966

###For Deployment
```
cd tnw
npm run build
npm run start
```
Runs on http://localhost:3000


## Linting

```
npm run lint
```
